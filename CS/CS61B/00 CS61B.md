- **Java Fundamentals:** syntax, classes, types, generics, testing, inheritance, comparison, iteration, and object methods.
- **Data Structures and Algorithms:** runtime analysis, lists, arrays, abstract data types, union-find, heaps, trees, hashing, graphs, and tries.
- **Sorting Algorithms:** comparison sorts, non-comparison sorts, stability, and runtime bounds.

All Java examples and data structure implementations are included directly in the relevant core note.
